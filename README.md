# Sales Totals Calculator

A lightweight browser-based calculator for quickly generating service and product totals, including labor charges, GST, PST, and final invoice totals.

## Features

* Calculate service charges based on minutes worked and hourly rate
* Add product charges to service totals
* Automatic GST and PST tax calculations
* Save custom tax rates and hourly rate to local storage
* Simple responsive UI using Pico CSS
* Reactive functionality powered by Alpine.js

## Tech Stack

* HTML5
* JavaScript
* Alpine.js
* Pico CSS

## Usage

1. Enter:

   * Minutes worked
   * Product charge

2. Optional:

   * Adjust hourly rate
   * Update GST/PST rates
   * Save settings for future use

3. Click **Calculate** to generate:

   * Service charge
   * Subtotal
   * GST
   * PST
   * Final total

## Local Storage Support

Saved settings include:

* Hourly Rate
* GST Rate
* PST Rate

These persist between sessions for faster repeat use.

## Purpose

Designed for freelancers, service providers, or small businesses needing quick and repeatable sales total calculations directly in the browser.
